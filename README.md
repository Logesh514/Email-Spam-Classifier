# Email Spam Classifier

## Objective
The objective of this project is to classify emails as Spam or Ham (Not Spam) using Machine Learning.

## Tools and Technologies
- Python
- Pandas
- Scikit-Learn
- Google Colab

## Dataset
Sample Email Dataset

## Algorithm Used
Multinomial Naive Bayes

## Features Used
- Email Text Content

## Project Workflow
1. Create email dataset
2. Convert text into numerical features
3. Split data into training and testing sets
4. Train the Naive Bayes model
5. Predict whether an email is Spam or Ham

## Result
The model successfully classified email messages as Spam or Ham.

## Output
Example Prediction:
- Input: "You have won a free prize"
- Output: Spam
